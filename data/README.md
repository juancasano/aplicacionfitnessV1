# Datos de ejemplo

En esta carpeta tienes CSVs de ejemplo que puedes subir a Supabase (Table Editor → Import data from CSV) o importar vía API.

Archivos:

- `users_sample.csv`
- `exercises_sample.csv`
- `workouts_sample.csv`
- `workout_exercises_sample.csv`

Instrucciones rápidas: en Supabase, abre Table Editor, selecciona la tabla destino y usa "Import data from CSV".
