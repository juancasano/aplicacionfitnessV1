# Módulo: Ejercicios - Especificación funcional

## Objetivo

Gestionar el catálogo de ejercicios: nombre, categoría, descripción y nivel.

## Funcionalidades

- Listar ejercicios.
- Crear, editar y eliminar ejercicios.
- Filtrar por categoría o nivel.
