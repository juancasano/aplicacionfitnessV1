# Módulo: Usuarios - Especificación funcional

## Objetivo

Gestionar usuarios: registro, edición y listado.

## Funcionalidades

- Listar usuarios.
- Crear y editar usuario (nombre, email).
- Eliminar usuario (restricciones en DB si existen workouts asociados).
